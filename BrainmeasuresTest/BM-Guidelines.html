Guidelines

Exam guidelines are a set of instructions and recommendations designed to ensure a fair and organized examination process.

These guidelines are crucial for creating a standardized and consistent environment for all students undergoing an examination. While specific guidelines may vary depending on the educational institution and the type of exam, the following points generally encompass key aspects of exam guidelines:

  • Coding semi-final
  • Coding finale
  • Hackathon finale
  • General

1

Only short-listed candidates will be able to access the problem statement.

2

The coding semi-final will have two coding problems. The duration of each problem is usually three hours.

3

The semi-final slot will open on as per schedule. The problem can be accessed on your Dashboard.

4

The application has a Live Coding Environment where the individual can write his/her code and compile the code.

5

Online webcam proctoring is mandatory to participate in the coding semi-final. The recorded video will be reviewed by the contest administrator and will be a key factor to declare the finalists.

6

To participate in the coding semi-final, the participant can use any of the following desktop browsers: Internet Explorer 10+, Google Chrome 53+, Mozilla Firefox 60+.

7

Semi-final problems can not be taken on mobile.

8

In case multiple attempts are allowed against each problem, the highest score of all the attempts will be considered as the final score.

9

Once the individual completes writing the code and compiles the code, a default test case will be used to check the code. Please note: No score is allotted for passing the default test case.

10

Ranking Criteria for Tiebreaker

Problem 1 (Time taken in Submission):

Developers are ranked according to the total score. Developers who have the same total score are ranked by least total time. Here total time is the sum of the time consumed for each problem solved.

Problem 2 (Early Submission)

In case two developers have taken the same time then whoever hits the leaderboard early, will be ranked higher.
